# 发布检查清单

## 📋 发布前检查

### 环境检查
- [ ] 运行 `./scripts/release/verify-scripts.sh` 通过
- [ ] GitHub CLI 已认证 (`gh auth status`)
- [ ] Android SDK 已配置（如需 Android 构建）
- [ ] 所有代码已提交到 Git

### 代码检查
- [ ] 更新版本号
- [ ] 更新 CHANGELOG.md
- [ ] 运行测试通过
- [ ] 代码审查完成
- [ ] 没有 TODO 或 FIXME 标记

### 构建检查
- [ ] 运行 `./scripts/release/test-build.sh` 成功
- [ ] Debug 版本构建成功
- [ ] Release 版本构建成功
- [ ] 构建产物大小合理

## 🧪 测试检查

### Android 测试
- [ ] 在真机上安装测试
- [ ] 应用启动正常
- [ ] 主要功能测试通过
- [ ] 状态栏显示正确（白色背景+深色图标）
- [ ] 没有崩溃或 ANR
- [ ] 网络请求正常
- [ ] 数据持久化正常

### iOS 测试（如适用）
- [ ] 在模拟器上测试
- [ ] 在真机上测试
- [ ] 状态栏显示正确（深色图标）
- [ ] 主要功能正常
- [ ] 没有崩溃

### Desktop 测试（如适用）
- [ ] macOS 测试
- [ ] Linux 测试（如适用）
- [ ] Windows 测试（如适用）
- [ ] 应用启动和运行正常

## 📦 构建检查

### 构建产物
- [ ] 所有平台构建成功
- [ ] 文件命名正确
- [ ] 文件大小合理
- [ ] 文件可以正常打开/安装

### Android 特定
- [ ] Debug APK 生成
- [ ] Release APK 生成
- [ ] AAB 生成（如需上架）
- [ ] APK 已签名（Release）
- [ ] 混淆配置正确

### Desktop 特定
- [ ] DMG 生成（macOS）
- [ ] DEB 生成（Linux）
- [ ] EXE 生成（Windows）
- [ ] 安装包可以正常安装

## 🚀 发布检查

### GitHub Release
- [ ] Release 创建成功
- [ ] 标签名称正确
- [ ] Release 标题和描述完整
- [ ] 所有构建产物已上传
- [ ] 下载链接可用
- [ ] Release Notes 完整

### 文档更新
- [ ] README.md 更新
- [ ] CHANGELOG.md 更新
- [ ] 版本号文档更新
- [ ] API 文档更新（如有）

## 📢 发布后检查

### 验证
- [ ] 从 GitHub Release 下载文件
- [ ] 验证下载的文件可以安装
- [ ] 验证安装后应用正常运行
- [ ] 检查 Release 页面显示正常

### 通知
- [ ] 通知团队成员
- [ ] 更新项目文档
- [ ] 发布公告（如需要）
- [ ] 更新应用商店（如适用）

### 监控
- [ ] 监控崩溃报告
- [ ] 监控用户反馈
- [ ] 监控下载量
- [ ] 准备热修复方案（如需要）

## 🔄 回滚计划

如果发布出现问题：

### 立即行动
- [ ] 停止推广新版本
- [ ] 评估问题严重程度
- [ ] 决定是否需要回滚

### 回滚步骤
- [ ] 删除有问题的 Release
- [ ] 恢复到上一个稳定版本
- [ ] 通知用户
- [ ] 修复问题
- [ ] 重新发布
