## 🚀 快速开始

### 步骤 1: 运行配置脚本
```bash
./setup-android-studio.sh
```

### 步骤 2: 重启 Android Studio
```
File → Exit
重新打开 Android Studio
```

### 步骤 3: 选择运行配置
```
顶部工具栏 → 运行配置下拉菜单 → 选择配置
```

### 步骤 4: 运行应用
```
点击 ▶️ 按钮 或 按 Shift + F10
```


## 🍎 iOS 配置

### 界面布局
```
┌────────────────────────────────────────────────────┐
│ [juejin-lite (iOS) ▼] [▶️] [⏹️]                   │
└────────────────────────────────────────────────────┘
     ↑                    ↑    ↑
  运行配置              编译 停止
```

### 使用步骤
1. **选择配置**: `juejin-main (iOS)` 或 `juejin-lite (iOS)`
2. **点击运行**: ▶️ 按钮（编译 Framework）
3. **查看输出**: Build 窗口
4. **Framework 位置**: `build/bin/iosSimulatorArm64/debugFramework/`

### 编译流程
```
┌─────────────────────────────────────────────────────┐
│  Android Studio                                     │
│  ├─ 点击运行                                        │
│  ├─ 编译 Kotlin 代码                                │
│  ├─ 生成 Framework                                  │
│  └─ 输出到 build/ 目录                              │
│                                                     │
│  ↓                                                  │
│                                                     │
│  Xcode                                              │
│  ├─ 添加 Framework                                  │
│  ├─ 运行 iOS 应用                                   │
│  └─ 调试和测试                                      │
└─────────────────────────────────────────────────────┘
```

### 注意事项
- ⚠️ 不能直接运行 iOS 应用
- ⚠️ 需要在 Xcode 中运行
- ✅ 可以编辑共享代码
- ✅ Framework 自动重新编译

## 🎨 代码编辑

### expect/actual 导航
```kotlin
// commonMain
expect fun getPlatformName(): String
         ↓ Ctrl + Alt + B
         
// androidMain
actual fun getPlatformName(): String = "Android"

// jvmMain (Desktop)
actual fun getPlatformName(): String = "Desktop"

// iosMain
actual fun getPlatformName(): String = "iOS"
```

### 跨平台代码提示
```
输入 "expect" → 自动提示创建 actual 实现
输入 "actual" → 自动关联 expect 声明
```
