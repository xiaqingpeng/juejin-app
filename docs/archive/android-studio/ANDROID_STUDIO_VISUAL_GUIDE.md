# 📱 Android Studio 可视化操作指南

## 🎯 目标

在 Android Studio 中添加并使用 `juejin-lite` 运行配置

### 1. 运行配置位置

```
│           ↑                 ↑               ↑   ↑   ↑      │
│           │                 │               │   │   │      │
│        运行配置          设备选择        运行 调试 停止    │
```

## 🚀 操作步骤（图文说明）

### 步骤 1：打开项目

```
1. 启动 Android Studio
2. 选择 "Open" 或 "Open an Existing Project"
3. 导航到项目目录：juejin-app
4. 点击 "Open"
```

### 步骤 2：同步 Gradle

```
┌────────────────────────────────────────┐
│  🔔 Gradle files have changed          │
│                                         │
│  [Sync Now]  [Dismiss]                 │
└────────────────────────────────────────┘
                ↑
            点击这里
```

或者手动同步：

```
┌────────────────────────────────────────┐
│ File  Edit  View  ...                  │
├────────────────────────────────────────┤
│  🐘 ← 点击这个图标（Sync Project）      │
└────────────────────────────────────────┘
```

### 步骤 3：查看运行配置

同步完成后，查看顶部工具栏：

```
┌────────────────────────────────────────┐
│  [juejin-lite ▼]  ← 应该能看到这个     │
└────────────────────────────────────────┘
```

如果看不到，点击下拉菜单：

```
┌─────────────────────────────┐
│ ▼ juejin-main              │
│   juejin-lite              │  ← 选择这个
└─────────────────────────────┘
```

### 步骤 4：选择设备

```
┌────────────────────────────────────────┐
│  [juejin-lite ▼]  [📱 设备 ▼]          │
│                    ↑                    │
│                点击这里                 │
└────────────────────────────────────────┘
```

设备列表：

```
┌─────────────────────────────────────┐
│ Running Devices                      │
│   📱 Pixel 5 API 33 (emulator-5554) │  ← 模拟器
│                                      │
│ Available Devices                    │
│   📱 Samsung Galaxy S21 (192.168..) │  ← 真机
│   ➕ Create New Virtual Device      │
└─────────────────────────────────────┘
```

### 步骤 5：运行应用

点击绿色运行按钮：

```
┌────────────────────────────────────────┐
│  [juejin-lite ▼]  [📱 Pixel 5 ▼]  ▶️  │
│                                    ↑   │
│                                点击这里 │
└────────────────────────────────────────┘
```

或使用快捷键：
- macOS: `Ctrl + R`
- Windows/Linux: `Shift + F10`

### 步骤 6：查看运行结果

底部会显示运行日志：

```
┌────────────────────────────────────────┐
│ Run ▼  Build  Logcat  Terminal  TODO  │
├────────────────────────────────────────┤
```

## 🔍 关键配置项说明

### Name（名称）
```
┌────────────────────────────┐
│ Name: juejin-lite          │  ← 显示在下拉菜单中的名称
└────────────────────────────┘
```

### Module（模块）
```
┌────────────────────────────────────────┐
│ Module: juejin-app.apps.juejin-lite.  │  ← 选择要运行的模块
│         main ▼                         │
└────────────────────────────────────────┘
```

模块列表：
```
┌─────────────────────────────────────┐
│ juejin-app.apps.juejin-main.main   │  ← 完整版

│ juejin-app.shared.core.common.main │

```

```
┌────────────────────────────┐
│ Launch: Default Activity ▼ │  ← 推荐使用
└────────────────────────────┘
```

选项：
```
┌─────────────────────────────┐
│ Default Activity           │  ← 推荐
│ Specified Activity         │
│ Nothing                    │

└─────────────────────────────┘
```

## 🎯 快速切换配置

### 方法 1：使用下拉菜单

```
点击 [juejin-lite ▼]
  ↓
选择 juejin-main
  ↓
点击 ▶️ 运行
```

### 方法 2：使用快捷键

```
按 Ctrl + Alt + R (macOS)
或 Alt + Shift + F10 (Windows/Linux)
  ↓
输入 "lite" 或 "main"
  ↓
按 Enter
```

### 方法 3：使用最近运行

```
按 Ctrl + R (macOS)
或 Shift + F10 (Windows/Linux)
  ↓
自动运行上次选择的配置
```

## 🎨 同时运行两个应用

### 步骤说明

```
1. 选择 juejin-main
   ↓
2. 点击 ▶️ 运行
   ↓
3. 等待应用启动
   ↓
4. 不要停止！
   ↓
5. 选择 juejin-lite
   ↓
6. 再次点击 ▶️ 运行
   ↓
7. 两个应用同时运行！
```

### 设备上的效果

## 💡 使用技巧

### 技巧 1：设置默认配置

```
右键点击配置名称
  ↓
选择 "Save 'juejin-lite' Configuration"
  ↓
下次打开项目自动选择
```

### 技巧 2：快速重新运行

```
按 Ctrl + R (macOS)
或 Shift + F10 (Windows/Linux)
  ↓
立即重新运行当前配置
```

### 技巧 3：停止应用

```
点击 ⏹️ 按钮
或按 Cmd + F2 (macOS)
或按 Ctrl + F2 (Windows/Linux)
```

### 技巧 4：查看构建时间

```

│ Build ▼                                │
├────────────────────────────────────────┤
│ BUILD SUCCESSFUL in 7s                 │  ← 这里显示时间
│ 170 actionable tasks: 21 executed      │

```
